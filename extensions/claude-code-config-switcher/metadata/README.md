# Metadata Screenshots

This folder contains screenshots for the Raycast Store listing.

## Requirements

- **Format:** PNG
- **Size:** 2000 x 1250 pixels (16:10 aspect ratio)
- **Quantity:** At least 3 screenshots (maximum 6)
- **Content:** Show the extension's key features and commands

## How to Generate Screenshots

### Using Raycast's Built-in Tool (Recommended)

1. Install and run the extension in Raycast (version 1.37.0+)
2. Open the command you want to screenshot
3. Press `⌘ + ⇧ + 2` (Cmd + Shift + 2) to capture using Raycast's Window Capture
4. Screenshots are automatically saved to this `metadata/` folder

### Suggested Screenshots

For this extension, capture:

1. **Manage Profiles Command** - Show the list of profiles with the active profile highlighted
2. **Switch Profile Command** - Show the quick profile switcher in action
3. **Create/Edit Profile Form** - Show the profile configuration interface
4. **Manage MCP Servers Command** (optional) - Show MCP server management

## Tips

- Use a clean, uncluttered background
- Show realistic data (but avoid sensitive information)
- Maintain consistent styling across all screenshots
- Focus on the extension's most useful features
- Use light mode unless dark mode is essential to demonstrate
